# Vercel Environment Variables Setup

## Required Environment Variables

These must be added to Vercel for the site to function correctly:

### Authentication & Security
```
SESSION_SECRET=<your-secret-key>
MAGIC_LINK_SECRET=<your-secret-key>
NEXT_PUBLIC_APP_URL=https://portal.concussion-education-australia.com
```

### Analytics
```
ADMIN_API_KEY=78007a9523265d8a53b15efc8457b60c2f3394d2d5ad0259a38761afb69a02d3
```

### Email (Resend)
```
RESEND_API_KEY=<your-resend-api-key>
```

### Storage (Vercel Blob)
```
BLOB_READ_WRITE_TOKEN=<auto-generated-by-vercel>
```

## How to Add Environment Variables to Vercel

1. Go to: https://vercel.com/concussionpros-projects/concussion-portal/settings/environment-variables

2. Add each variable:
   - Click "Add New"
   - Enter Variable Name
   - Enter Value
   - Select Environment: Production, Preview, Development (select all)
   - Click "Save"

3. After adding all variables, redeploy:
   - Go to Deployments tab
   - Click "..." on latest deployment
   - Click "Redeploy"
