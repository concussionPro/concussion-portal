# Module Videos

This directory should contain the video files for each learning module.

## Required Videos

1. `module-1-concussion-science.mp4` - What is a Concussion? (The Science & Mechanisms)
2. `module-2-red-flags.mp4` - Identifying Red Flags (Clinical Signs)
3. `module-3-management-recovery.mp4` - Management & Recovery (Return to Play/Work)
4. `module-4-special-topics.mp4` - Special Topics (Aged Care, Pediatrics, MVA)

## Video Requirements

- Format: MP4 (H.264 codec recommended)
- Resolution: 1080p recommended, 720p minimum
- Duration: See module data for recommended durations
- Ensure videos are optimized for web playback

## CPD Compliance

Each module has a minimum required viewing time for CPD compliance:
- Module 1: 15 minutes
- Module 2: 12 minutes
- Module 3: 20 minutes
- Module 4: 18 minutes

The video player tracks watch time and enforces these requirements before allowing module completion.

## Placeholder Videos

For development/testing, you can use placeholder videos or demo content. The application will gracefully handle missing videos with an error message.
